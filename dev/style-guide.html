Logo Usage Guidelines

  • Always maintain the aspect ratio - never stretch or distort the logo
  • Ensure adequate clear space around the logo (minimum 30% of logo width)
  • Use the purple color #AF52DE for the glasses in the icon
  • Primary logo colors are Off-Black #1C1C1E and Purple #AF52DE
  • For dark backgrounds, use inverted (white) versions of the logo
  • Minimum size: 32px height for icon only, 120px width for full logos
  • Prefer horizontal layout for web headers and documents
  • Use vertical layout for mobile splash screens and social media

üéØ 3D Shadow Technique

Implementation Pattern

/* HTML Structure */ <div class="wrapper"> <element class="main-element">Content</element> </div> /* CSS Implementation */ .wrapper { position: relative; display: inline-block; transition: transform 0.2s; /* For hover animations */ } .wrapper::before { content: ''; position: absolute; top: 6px; /* Offset down */ left: 0; right: 0; bottom: -6px; background: #E5E5EA; /* Light Gray for shadows */ border: 2px solid #1C1C1E; /* Off-Black */ border-radius: 12px; /* Match main element */ z-index: 1; pointer-events: none; } .main-element { background: white; border: 2px solid #1C1C1E; /* Off-Black */ border-radius: 12px; position: relative; z-index: 2; /* Your styles here */ } /* Hover effect - move the entire wrapper */ .wrapper:hover { transform: translateY(-2px); }

This is a card with the 3D shadow effect applied using the wrapper technique.

Key Points

  • Shadow offset is 6px down only (no horizontal offset)
  • Both element and shadow have 2px Off-Black (#1C1C1E) border
  • Borders perfectly align on left and right
  • Shadow background is #E5E5EA (Light Gray for shadows only)
  • Use wrapper::before for the shadow layer
  • Main element gets z-index: 2, shadow gets z-index: 1
  • CRITICAL: Apply hover transforms to the wrapper, not the element, so shadows move in sync
  • Use transition: transform 0.2s on wrappers for smooth animations

‚ú® Animations & Interactions

Animation Principles

  • Wrapper-based transforms: Always apply hover transforms to wrapper elements containing shadows
  • Consistent timing: Use 0.2s transitions for responsive, snappy interactions
  • Synchronized movement: Shadows must move with their parent elements
  • Smooth scrolling: Use CSS scroll-behavior: smooth for navigation links

Standard Hover Heights

/* Small lift - buttons, nav items */ transform: translateY(-2px); /* Medium lift - cards, tiles */ transform: translateY(-4px); /* With rotation - game tiles */ transform: translateY(-4px) rotate(-1deg); /* odd children */ transform: translateY(-4px) rotate(1deg); /* even children */

Timing Functions

--transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1); --transition-fast: all 0.2s ease; /* Preferred for hover states */ --transition-slow: all 0.5s cubic-bezier(0.4, 0, 0.2, 1);

Common Animations

/* Bounce animation */ @keyframes bounce { 0%, 100% { transform: translateY(0); } 50% { transform: translateY(-10px); } } .bounce-animation { animation: bounce 2s infinite; } /* Pulse animation */ @keyframes pulse { 0% { transform: scale(1); } 50% { transform: scale(1.05); } 100% { transform: scale(1); } } .pulse-animation { animation: pulse 2s infinite; } /* Loading shimmer */ @keyframes loading { 0% { background-position: 200% 0; } 100% { background-position: -200% 0; } } .loading { background: linear-gradient(90deg, #f0f0f0 25%, #e0e0e0 50%, #f0f0f0 75%); background-size: 200% 100%; animation: loading 1.5s infinite; }
